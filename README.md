## ğŸ“– Overview

**YouTube Speed Toggle** is a lightweight browser extension that injects a native-looking speed control button directly into the YouTube player control bar. 

YouTube's default speed setting requires **two clicks and a menu navigation**. This extension solves that friction by allowing you to cycle through playback speeds with a **single click**. Perfect for students, tutorial watchers, and power users.

5.  Select the `manifest.json` file from the folder.
