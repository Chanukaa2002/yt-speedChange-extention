# YouTube Speed Toggle Extension

A lightweight browser extension that adds a playback speed control button to the YouTube video player.

## Features

- **One-Click Speed Control**: Cycle through speeds: `0.25x`, `0.5x`, `0.75x`, `1x`, `1.25x`, `1.5x`, `1.75x`, `2x`, `2.5x`, `3x`.
- **Integrated UI**: Seamlessly blends with the YouTube player control bar.
- **Syncs**: Automatically updates if you change speed via keyboard shortcuts or settings.

## Installation

### Chrome / Edge / Brave

1. Go to `chrome://extensions`.
2. Enable **Developer mode** (toggle in top-right).
3. Click **Load unpacked**.
4. Select this folder.

### Firefox

1. Go to `about:debugging`.
2. Click **This Firefox** on the left.
3. Click **Load Temporary Add-on...**.
4. Select the `manifest.json` file in this folder.

## Usage

Simply click the speed button (e.g., "1x") in the YouTube player control bar to cycle to the next speed.
